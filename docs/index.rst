aladhan.py
==========
aladhan.py is an asynchronous pythonic wrapper for the `Aladhan prayer times <https://aladhan.com/prayer-times-api>`_ API.

Getting started
---------------

Is this your first time using the framweork? This is the place to get started!

- **First steps:** :ref:`intro`
- **Examples:** Many examples are available in the :resource:`repository <examples>`.
